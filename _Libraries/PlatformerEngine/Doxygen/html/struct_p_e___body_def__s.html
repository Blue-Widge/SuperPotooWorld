Référence de la structure PE_BodyDef_s
Fonctions relatives à un corps

Structure regroupant les informations nécessaires à la création d'un corps. Une même définition peut servir à créer plusieurs corps. Plus de détails...

#include <PE_Body.h>

Champs de données

PE_BodyType type
 Le type du corps.
 
PE_Vec2 position
 La position du corps exprimée dans le référentiel monde.
 
PE_Vec2 velocity
 La vitesse linéaire de l'origine du corps exprimée dans le référentiel monde.
 
float xDamping
 L'amortissement linéaire du corps sur l'axe des abscisses. Ce paramètre permet de réduire la vitesse linéaire horizontale du corps.
 
float yDamping
 L'amortissement linéaire du corps sur l'axe des ordonnées. Ce paramètre permet de réduire la vitesse linéaire verticale du corps.
 
float gravityScale
 L'échelle de gravité du corps.
 
void * userData
 Pointeur vers les données de l'utilisateur. La librairie ne le modifie pas. Il est défini à NULL par défaut.
