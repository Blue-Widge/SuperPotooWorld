Attributs protégés

PE_Node m_worldNode
 Noeud contenant un pointeur vers ce corps dans la listes des corps du monde parent. Ce membre appartient au monde parent.
 
PE_BodyType m_type
 Type du corps.
 
int m_flags
 Flags du corps.
 
PE_Vec2 m_position
 Position du corps exprimée dans le référentiel monde.
 
PE_Vec2 m_velocity
 Vitesse linéaire de l'origine du corps exprimée dans le référentiel monde.
 
PE_Vec2 m_forces
 Somme des forces appliquées au corps, exprimée dans le référentiel monde.
 
float m_xDamping
 Amortissement linéaire du corps sur l'axe des abscisses. Ce paramètre permet de réduire la vitesse linéaire horizontale du corps.
 
float m_yDamping
 Amortissement linéaire du corps sur l'axe des ordonnées. Ce paramètre permet de réduire la vitesse linéaire verticale du corps.
 
float m_gravityScale
 Echelle de gravité du corps.
 
PE_World * m_world
 Pointeur vers le monde parent de ce corps.
 
PE_List m_colliderList
 Liste des colliders attachés à ce corps.
 
PE_Sweep m_sweep
 Décrit le mouvement du corps pendant un pas d'intégration.
 
PE_Sleep * m_sleep
 Etat d'endormissment du corps.
 
PE_SolverBody * m_solverBody
 Pointeur vers les données du solver associées à ce corps.
 
void * m_userData
 Pointeur vers les données de l'utilisateur. La librairie ne le modifie pas. Il est défini à NULL par défaut.
