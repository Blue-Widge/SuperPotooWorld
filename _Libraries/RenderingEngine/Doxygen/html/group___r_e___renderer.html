◆ RE_Color_set()

RE_API RE_Color RE_Color_set ( int  r,
int  g,
int  b,
int  a 
)

Construit une couleur à partir de ses composantes [R,V,B,A] chacune entre 0 et 255.

Paramètres
[in]rl'intensité du rouge.
[in]gl'intensité du vert.
[in]bl'intensité du bleu.
[in]ala composante alpha (transparence) de la couleur. La couleur est parfaitement transparante quand elle vaut 0 et elle est opaque quand elle vaut 255.
Renvoie
La couleur spécifiée.
