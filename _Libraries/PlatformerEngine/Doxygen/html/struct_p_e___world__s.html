Attributs protégés

PE_Allocator * m_allocator
 Allocateur principal utilisé pour toutes les allocations mémoire dans le monde.
 
float m_timeStep
 Pas de temps utilisé pour chaque mise à jour du monde. Le pas de temps est fixe et varie typiquement entre 1.f/50.f et 1.f/60.f.
 
PE_Vec2 m_gravity
 Vecteur de gravité global du monde. Note that the gravity of a body can be scaled.
 
PE_List m_bodyList
 Liste des corps du monde.
 
PE_CollisionDetector * m_collisionDetector
 Détecteur de collisions.
 
PE_Solver * m_solver
 Solver utilisé pour résoudre toutes les collisions dans ce monde.
 
Bool m_locked
 Booléen permettant de bloquer l'accès à certaines fonctions lors de callback.
