Référence de la structure PE_ColliderDef_s
Fonctions relatives à un Collider

Structure regroupant les informations nécessaires à la création d'un collider. Une même définition peut servir à créer plusieurs colliders. Plus de détails...

#include <PE_Collider.h>

Champs de données

PE_Shape shape
 Forme du collider.
 
Bool isTrigger
 Indique si le collider est un déclencheur.
 
Bool isOneWay
 Indique si le collider a un comportement "one-way". Autrement dit, s'il autorise uniquement les collisions dans une seule direction.
 
PE_Filter filter
 Filtre à appliquer au collider pendant la détection des collisions.
 
void * userData
 Pointeur vers les données de l'utilisateur. La librairie ne le modifie pas. Il est défini à NULL par défaut.
